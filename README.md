幸运大转盘抽奖系统
 
一个功能完整的动态抽奖转盘，支持从表格读取账号、动态扣减抽奖次数，且转盘精准对齐，指哪中哪。
 
🎯 核心功能
 
- 动态抽奖：支持先快后慢的转动效果，体验更真实
​
- 账号管理：从  lottery_config.xlsx  表格读取账号和剩余次数
​
- 精准对齐：转盘分区精准校准，指针指向哪个奖品就抽中哪个
​
- 黑金配色：视觉上与PPT静态转盘完美统一
 
🚀 快速使用
 
1. 运行脚本：在 PyDroid 3 中打开并运行  彩票轮盘.py 
​
2. 获取页面：脚本运行成功后，在手机下载文件夹找到生成的  彩票轮盘.html 
​
3. 开始抽奖：用浏览器打开  彩票轮盘.html ，输入账号即可开始抽奖
 
📁 项目文件
 
-  彩票轮盘.py ：核心生成脚本
​
-  彩票轮盘.html ：动态抽奖页面
​
-  lottery_config.xlsx ：账号与次数配置表
